# remahini.github.io
Event-related potential (ERP) techniques are a powerful tool for studying cognitive, motor, and sensory processing of brain functions. It is also well-known that all brain works, and information processing depends on brain responses at a specific time window. Here, we design a multi-subject consensus clustering from multiple-clustering results to explore reliable pieces of information shared from a set of similar sources (e.g., subjects' brains). A time window determination method has been designed using the idea of estimating stable electrode configuration in milliseconds of consecutive time bins in the perfect cluster map (i.e., cluster with high inner-similarity). Therefore, all the time points in the time window for identifying the ERP component share the same spatial (topographical) and temporal dynamics from the multiple subjects. Our findings reveal that the temporal and spatial dynamics of the brain response can be reliably explored using the proposed methodology, especially when there is little information available about measuring the cognitive process .
